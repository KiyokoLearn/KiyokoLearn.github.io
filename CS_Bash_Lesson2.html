Lesson 1 -- Moving Around


To begin using command line, you must first open up the terminal. If you are unsure how to do that, you can visit the tutorial under the Resources Page here.

Technical terms you should know before you begin:
Technical Term Translation
Directory Folder
GUI (Graphical User Interface) Everything that is interactable on the screen. The icons pictures of folder, files, etc.
Working Directory The folder you are currently in
Pathname The directions take to get to a certain directory from the systems starting point.


Once you are in the terminal, you will see the prompt, which looks something like this:
Username@ComputerName:~$


Right now, you are in your Username directory. When using a GUI, you would see a little window, typically with a bunch of pictures of folders labeled desktop, documents, pictures, etc. So, if I'm in the home directory, why don't I see all those folders? If you type in "ls" (which stands for list), it will print on the screen everything that is in that directory.

After the command prompt, simply type ls:
Username@ComputerName:~$ ls


Then press enter. It should look something like this:

backup       documents       images       videos      
bin       documents       templates       downloads      
music       tmp      


It is ok, if you have more or less folders than the ones I listed above.

But what if I want to go into one of those folders? When using a normal GUI, you would simply double click on the folder (or single click depending on your settings). In the termainal, you have to use the command "cd" (change directory). You would type "cd" and then the name of the folder you want to go into.

For example, to go into the documents folder type:
Username@ComputerName:~$ cd documents/


When you press enter, your prompt will change to:
Username@ComputerName:~/documents$


The change in the prompt shows that we are now in the documents directory. If we want to see what we have in this folder, you can use the ls command again.
Username@ComputerName:~/documents$ ls


In my documents folder, I have files and folders, most of which are school related.
Essay.txt       Kiyoko_Learn       Me.jpg      


Depending on your settings, these may be color coded. On my particular terminal, the image (Me.jpg) is purple, the folder (Kiyoko Learn) is blue, and the text file (Essay.txt) is white.

Now, what if I don't want to be in the documents folder anymore? What if I want to go back to the home folder? What if I want to hit the "back botton" per say? To do this, use the "cd .." command. Literally this means change the working directory to the one above the current directory.

For example, if I use type in the "cd .." command:
Username@ComputerName:~/documents$ cd ..


When I press enter, my promt changes back to:
Username@ComputerName:~$


Now I've been changing directories a lot and I've gotten really far from my Username directory. I don't feel like typing "cd .." over and over again until I get back to where I started. Is there a shortcut for this? Of course there is! use the command "cd ~". The ~ stands for your Username.
Username@ComputerName:~/Folder1/Folder2/Folder3$ cd ~


So, I like the directories I have and all, but what if I want to make a new one? The command "mkdir" stands for make directory. You would type "mkdir" and then the name of the directory.

For example, if I wanted to make a new directory called Practice:
Username@ComputerName:~$ mkdir Practice

When you press enter, your prompt will simply appear again and I'm sure your first response will be, "What just happened? Was my directory made or not?" Rest assured, there is nothing to fear. If you type in "ls" again, your directory should be there.

Username@ComputerName:~$ ls
Essay.txt       Kiyoko_Learn       Me.jpg       Practice      


So I've been moving around directories a lot, and now I'm really confused where I am. If you look at the prompt, that should give you and idea of where you are. But what if I want to know the full pathname? Use the "pwd" command which stands for print working directory. For example, if I am in the new folder Practice that we just created and I wanted to see the full pathname: Username@ComputerName:~/Practice$ pwd
/home/Username/Practice


Other details you don't care for:
$ ls -a "List all" - View hidden files. Hidden files will be denoted with a . before them
$ ls -l "Long list" - Lists name of file, when it was last modified, and its size in bytes.
